# 可用模型

## Claude 模型

| 模型 ID | 描述 |
| ---------------------------- | ---------------------------------------- |
| `claude-sonnet-4-5-thinking` | Claude Sonnet 4.5（支持扩展思考） |
| `claude-opus-4-5-thinking` | Claude Opus 4.5（支持扩展思考） |
| `claude-sonnet-4-5` | Claude Sonnet 4.5（无思考功能） |

## Gemini 模型

| 模型 ID | 描述 |
| ------------------- | ------------------------------- |
| `gemini-3-flash` | Gemini 3 Flash（支持思考） |
| `gemini-3-pro-low` | Gemini 3 Pro Low（支持思考） |
| `gemini-3-pro-high` | Gemini 3 Pro High（支持思考） |

Gemini 模型包含完整的思考支持，并处理多轮对话中的 `thoughtSignature`。
