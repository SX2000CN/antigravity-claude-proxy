# macOS 菜单栏应用

对于喜欢原生体验的 macOS 用户，我们提供了一个配套的菜单栏应用，无需通过终端即可快速控制服务器。下载地址：[antigravity-claude-proxy-bar](https://github.com/IrvanFza/antigravity-claude-proxy-bar)

> **注意：** 这只是一个 GUI 包装器。你仍然需要先使用上述 [安装方法](../README.md#installation) 之一安装并设置代理服务器。

![AntiGravity Claude Proxy Bar](https://github.com/IrvanFza/antigravity-claude-proxy-bar/blob/main/images/application.png?raw=true)

## 主要功能

- **服务器控制**：一键启动/停止代理服务器，或使用 ⌘S 快捷键。
- **状态指示器**：菜单栏图标一目了然地显示服务器运行状态。
- **WebUI 访问**：直接从菜单打开 Web 管理控制台。
- **端口配置**：自定义代理服务器端口（默认为 8080）。
- **账号选择策略**：在 Hybrid（混合）、Sticky（粘性）或 Round-Robin（轮询）负载均衡策略之间选择。
- **自动启动选项**：应用启动时启动服务器，以及登录时启动应用。
- **原生体验**：专为 macOS 设计的简洁原生 SwiftUI 界面。
